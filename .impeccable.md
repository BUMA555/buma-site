# Impeccable Design Context

## Target audience
- 一人公司创业者
- 1-5 人小团队负责人
- 想把官网、内容、AI 团队、飞书消息、咨询承接接成业务闭环的人
- 需要先看懂、再信任、再联系的人

## Primary use cases
- 快速理解：一个人的公司也能拥有一支 AI 团队
- 评估是否适合用 OpenClaw / 飞书 / 官网内容链路来提升承接效率
- 找到最短阅读路径：文章中心、FAQ、解决方案、联系页
- 带着真实问题直接咨询，而不是在站内迷路

## Brand personality / tone
- 高级、克制、利落
- 更像海外科技公司官网，而不是本地服务站模板
- 少废话、少装饰、少堆卡片
- 强调信任感、秩序感、执行力
- 不是炫技型未来感，也不是廉价 AI 感

## Visual direction
- 深色系为主，偏冷静、专业、现代
- 首页首屏要有强主视觉，但不能廉价、不能模板味
- 模块节奏要清楚，留白要充足
- 允许结构级借鉴参考站，但不能抄成四不像

## Anti-patterns to avoid
- 卡片套卡片
- 首屏信息堆砌
- 乱用渐变、发光、玻璃拟态
- 普通示意图硬充门面
- 顶部堆电话、微信、按钮、说明，导致很挤
- 只有 logo 没有正文相关配图
- 图不相关、审美廉价、没有品牌调性

## Current hard rules
- 首页先过关，再扩其他页面
- 文章必须图文并茂，图片必须和主题相关
- 首页大图必须高质感、适合品牌调性
- 没上线 = 没完成
- 用户看不到 = 没完成
