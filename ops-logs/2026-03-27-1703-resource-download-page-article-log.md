## 提炼出的高质量写法
### 标题角度
- 常见角度是 `examples + best practices`
- 或 `definition + how to create`
- 或 `before you design`
- 共同点：先给结果，再讲结构

### 开头写法
- 先回答“这页能让我立刻得到什么”
- 强调单一目标，不先讲大理论
- 多用即时收益词：download / guide / template / checklist / examples

### 模块结构
- Hero
- 资源预览 / 截图 / 目录
- 你将获得什么
- 适合谁 / 不适合谁
- 低摩擦表单
- 信任信息 / 证据
- FAQ
- 下载后的 next step

### 转化 CTA
- 单一 CTA 最常见：`Download Now` / `Get the Guide` / `Get Template`
- 不推荐同屏多个入口抢路径

### 关键词覆盖
- `lead magnet`
- `template`
- `download`
- `checklist`
- `landing page`
- `high-converting`
- `examples`
